# Ethereal Forest

손그림 일러스트 기반 인터랙티브 3D 웹사이트 (Vite + Three.js + GSAP)

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 으로 접속합니다.

## 조작

- **방향키 / 휠**: 카메라 이동·줌
- **Enter**: 나비 커서 위치에서 클릭
- **Space**: 나비 커서 순환 / 카드 뒤집기
- **Backspace**: 뒤로가기

## 빌드

```bash
npm run build
npm run preview
```
